Introduction

PhotoMasque makes it possible to create advanced effects of contour and transparency on your
images by using preset masks. It's also possible to create personalized masks. The images used
in the masks are all in grayscale. White is the color of transparency and levels of gray represent
opacity. The darker the gray, the more opaque will be the color applied. PhotoMasque comes with
several base masks, use them to inspire you to create your own masks.


 Choosing the mask

 First, click on [Filter > PhotoMasque], then click on the small opening file folder. Masks can be made
from the most common formats recognized (JPG, GIF, png, BMP, TGA, PSP, PSD). After selectiong
the mask, a preview is shown in the corresponding window. Your selection will be preserved for the
next time you use PhotoMasque. It's a good idea at this point to set the orientation of the mask
according to the orientation of the image by using the rotation button, which is below the preview.
This will minimize mask deformations and optimize the results .


Parameters

 Color

This property only defines the color of the mask for the Color mode.
Click on the rectangle coloured to modify it.

Opacity

Allows you to modify the opacity of the mask in order to carry out effects of semi-transparency.
This option is not available in Transparent mode.

Mode

• Color
Allows you to set the color and opacity of the mask.

• Transparency
The mask is used to defines automatically the transparency of the image or of the layer.

• Grayscale
Allows the mask to be used to create grayscale effects.

• Gaussian blur
Allows the mask to be used to create gaussian blur (very fuzzy) effects.

 Blur

Allows you to smooth the mask before applying it.

Duatone

This option cancels the levels of gray. Pixels are rounded to the nearest black or white color.
The grayscale is thus removed.

Stretch mask

The size of the mask is adjusted by deforming it to that of the image.

Tile mask

Size of the mask is not modified. As many copies of the mask as are necessary are used to
cover the surface with the image.


Saving

It is important to note that currently only GIF and PNG formats preserve transparency. Please choose
these formats so as not to have nasty surprises the next time you open the image. In these cases,
the number of colors is small (256) with a method of diffusion.


Creation of personalized masks

Your masks should be made in standard 256 grey colors. It is preferable to use a format supporting the
color mode indexes (GIF, PNG, BMP, TIFF) in order to conserve the original quality of the mask without
loss of information, particularly if the latter contains degraded content. But, it is possible to use the
JPEG format with a compression ratio sufficient for limiting loss due to compression.


Tricks and easy ways

In Color mode, apply several layers of the same mask with a different color (blue sky, then white with the
Aquarelle mask for example). The combination of several masks gives sometimes an interesting result.
Test the mask "Dilution" then the mask "Watercolor" with a white color.
