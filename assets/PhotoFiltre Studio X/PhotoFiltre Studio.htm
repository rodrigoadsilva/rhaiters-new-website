Availability of filters and menus

The filters in certain menus are available only in 16 million colors without the transparency option
because most of them recompute and modify the colors. For example, they are disabled for GIF
images and images in indexed colors mode. If you do not have access to these menu item filters,
perform the following operations:

• Select the menu item "Image / Transparent color" (in situation of an image with a transparent color)
  then click on the option "Disable transparency"
• Select the menu item "Image / Mode / RGB color"


Image adjustment

The buttons on the tool bar make it possible to make precision adjustments of the luminosity,
contrast and saturation of colors. If your image is too dark or too clear, you must first carry out
a corresponding gamma correction using the menu or buttons.


Selection tools

PhotoFiltre Studio has two types of selection. The first type gathers the automatic forms (rectangle,
ellipse, triangle, rhombus). The second type corresponds to the lasso modes and polygon.
They both allow defining a form personalized by drawing contour or by a succession of lines.
Each selection can be safeguarded in a file to be re-used later on via the "Selection" menu.
PhotoFiltre Studio also manages magic wand selections type but they are not treated in vectorial
mode.

The menu tem "Edit / Stroke and fill" makes it possible to draw and fill these forms in the image.
You can also apply symmetries and rotations to the selection using the menu
"Selection / Transformation".

Some forms of preset selections (stars, arrows...) are available on my web site.


Tool pallet

This consists of drawing tools such as the pipette (color sampler), scrolling tool, line tool, fill tool,
spray tool, paintbrush, advanced paintbrush, blur tool and clone stamp.

PhotoFiltre Studio uses standard shapes (circles and squares of various sizes) but also more varied
forms (oblique line, sheet, star...)

Most of these functions are available only in 16 million colors mode.


Text insertion

The text module uses several parameters which modify the appearance of the result. For 90
degrees or -90 degrees text rotation, it's better to use menu "Edit / Transform" if your system
doesn't manage smoothing (Win95 and Win98). You can also create mirror effects with the
horizontal or vertical symmetry functions, available in this same menu. Certain options such as
outer bevel and blur optimize results when using a larger text size.

As long as the text is not validated (menu item "Edit / Validate text") you can move it with the
mouse and modify it (double-click or select the menu item "Edit / Text options").


Merging images

Here are steps to easily merge images :
• Open the two images
• Copy one of two images (CTRL+C)
• Switch to the other image
• Select "Edit / Paste special / Assemble" and choose the position for the clipboard contents

You can use this method to place images end to end in order to create a panorama.


Image dissolves

To do an image dissolve, simply :
• Open the two images
• Copy the smaller image into the larger. The paste create automatically a new layer
• Select the menu item "Layer / Options" and modify opacity
• Position the new layer
• Select the part of the resulting image which you want to preserve
• Select the menu item "Image / Crop"
• Select the menu item "Layer / Merge all visible layers"


Red eye reduction

• Begin by zooming on the image in order to be able to select the pupil easily
  (values of 400% or 800% should work)
• Use the "Retouch tool" in the tool palett and select the "Red eye" option in the list
With this tool, draw on the red part of the pupil to reduce the red value
• Repeat the process on the second pupil


 Selecting a transparent color

Transparency is available in indexed colors mode and in RGB color mode.

So to define a transparency color, follow these steps :
• Select the menu item "Image / Mode / Indexed color"
• Select the menu item "Image / Transparency color"
• While the "Transparent color" window is open, select the color to be transparent by clicking on it in
  the image by using the pipette (the cursor changes to the pipette automatically while doing this)


 Import formats and transparency

PhotoFiltre Studio supports the most widespread formats, however transparency is available only for
the following :
• GIF images
• PNG images in indexed colors mode
• Icons


Export formats

Automation

The menu item "Tools / Automate / Batch" makes it possible to carry out a succession of operations
in an automatic way to a batch or a selection of files.

"Files" tab

• Select the files source and the output destination
• Select the export format
• You can rename the files in sequence (optional)
• A reduction of the number of colors and adjustment of image quality (depending on image
  output format) are also available
• By checking "Custom selection", it's possible to carry out a manual selection of the images
  to be treated

"Image" tab

All the options are optional.
The option "Canvas size" is a little complex. It makes it possible to add blank space around your
image by imposing a width and/or a height increase. The calculation of the width and height will be
automatic if you leave a zero value (to add only one shade for example).

"Adjust" tab

Allows you to adjustment bases (brightness, contrast, saturation, gamma).
All the options are optional.
To convert a series of images into level of gray, adjust saturation at least (-100%).

"Filter" tab

Allows to apply some preset filters.
All the options are optional.

"Transform" tab

Allows to carry out rotations and flips.
All the options are optional.

"Copyright" tab

Allows to apply a copyright in all treated image.

"Action" tab

Start the process and display report.
To cancel the process, press Escape key and wait until the program returns. This action can take
one or several seconds depending on your computer's speed. In this tab, you can also save your
parameters.


PhotoMasque module

This module is available in 16 million colors only (RGB mode) via the menu "Filter / PhotoMasque".
It makes it possible to carry out effects of advanced contour and transparency using preset masks.
To learn some more about this modulen click here.


External plug-ins

PhotoFiltre Studio manages several type of external modules. Each type is placed in a dedicated menu.
The files are with the format "pfl" (PhotoFiltre Library) and must be installed in the "Plugins" folder.

Filter plug-ins
Available only for images in 16 million colors mode and apply to the selection of the image in progress.
They are listed under the menu "Filter / Plug-in".

Image plug-ins
May be limited by the number of colors and the transparency, and apply to the whole image in
progress. They are listed under the menu "Image / Plug-in".

Importation plug-ins
Allow importation of various formats and are always available (with or without image).
They are listed under the menu "File / Import".

Exportation plug-ins
 Allow export of the image in progress to various formats.
They are listed under the menu "File / Export".

Tool plug-ins
Intended for functions which cannot be classified in the types above.
They are listed under the menu "Tools / Plug-in".

Photoshop 8BF plug-ins
Available only for images in 16 million colors mode and apply to the selection of the image in progress.
They are listed under the menu "Filter / Photoshop 8BF".
